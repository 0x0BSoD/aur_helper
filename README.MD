# Little AUR Helper "ah"

Required requests and tabulate

```
usage: ah.py [-h] [-s SORT] {install,search,info} package_name

positional arguments:
  {install,search,info}
                        Awailable actions
  package_name          Name package placed in AUR

optional arguments:
  -h, --help            show this help message and exit
  -s SORT, --sort SORT  For search, sort by 0 - Name, 2 - Update date, 3 -
                        Votes
```
